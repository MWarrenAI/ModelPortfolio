Blender



Figure 10 A set of rendered objects made using Blender (Rendered by Mark Warren)


Blender allows for the crafting and modelling of various objects that can be used for things such as animation, game characters or moulding clay to be printed out later on. We made use of this to demonstrate the concept of our Head model and modifying models that would otherwise be too simple in Fusion 360. We found that this software was not really useful for scaling the model but more for the purposes of detailing model characteristics rather than exact measurements for printing the head via a 3D printer this would allow us to “mould” our base to better suit our model.


The disadvantage would be that due to how complicated the model is you’d have to simplify the model so that you don’t end up with too much geometry to print out. This requires a lot of time as opposed to Fusion 360 and following a lot of tutorials to get the exact result you want out of the model. For these reasons I’d only recommend this tool for use of editing models that needs detail to be manipulated rather than just “scaled”. I found that you could scale a model down but only by using the desired multiplication of the scale you desire such as 50% or .5 after you select an object press s and type the desired value. A much more accurate way of scaling I’d recommend Fusion 360 for as I found the software to be rather complicated to use in comparison.


You can render objects made in blender to provide image files such as the ones below – this can be useful for visual concepts. You can click on these images to show the full view of these rendered images. Camera and lighting effects allow you to enhance the image whereas otherwise the image would lack shading and textures hence the difference between just a STL file and a rendered image.


An Industrial example of Blender would be using Blender software to develop live 3D models of animals for use in an interactive encyclopaedia (Khan, S. et al, 2019).
