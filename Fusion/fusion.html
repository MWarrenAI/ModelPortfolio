Fusion 360


Autodesk Fusion 360 is free software for students and home users, the tool will become an expense when the software is used commercially. Autodesk is useful for printing out objects using a 3D printer, this piece of software is known to be for 3D cloud based CAD, CAM, CAE and PCB professional product design and manufacturing. This allows for products to have ensured aesthetics, form fit and function (Autodesk Inc, 2023). A useful video demonstration has been used to inform myself of the following below Fusion 360 information (McWhorter, P., 2017).



We made use of Fusion 360 to model the Head Hinges and Eyes of our model – this was particularly useful in order to accurately shape and model parts that can be manufactured and manipulated using screws and motors to control the hinges and structure of our head model. Unfortunately due to time constraints we did not make use of the eyes for our model.


An industrial example of Fusion 360 would be modelling a digital framework of a Quadcopter for use in Simulation to test the intensity and other properties of the device (Shelare, S. et al, 2021).

Create Sketch – allows us to switch to a Sketch mode to create 3d bodies in which we can manipulate object design via Extrude, Revolve, Loft, etc.



Figure 17: A Screenshot of Create Dropdown menu (Screenshot taken by Mark Warren)


The create drop down allows us to select different tools in which we can improve the dimensions of our sketch to fit the shape or tool we want for the purposes of design.



Figure 18: A Screenshot of the Canvas demonstrating the Dimension Tool (Screenshot taken by Mark Warren)


We can make use of the Dimension tool to set the dimension we would want for length. This is accessed via pressing D and hovering the cursor some space away from the object drawn. This allows us to accurately represent how long a dimension is without having to draw this out physically otherwise with a pen and paper carefully ensuring the dimensions match via metric units.



Figure 19: A Screenshot of the Select Tool (Screenshot taken by Mark Warren)


Select allows us to select objects drawn onto the existing plane.



Figure 20: A Screenshot of the Sketch Profile Cube Tool (Screenshot taken by Mark Warren)


Sketch Profile cube allows us to change the perspective or orientation of the plane.



Figure 21: A Screenshot of the Extrude Tool (Screenshot taken by Mark Warren)


Extrude allows us to add depth to our objects.

Pros


  • Familiar feeling to tools like Blender.
  • Software is more exact than other Microsoft Tools meaning less time is wasted on precision.
  • Could be used to quickly visualise a design for a product.
  • Professional Software that is offered for free for most users.

Cons


  • Tool can be limited to what is offered via the free software.
  • Can be Complicated and requires a large amount of time to grasp if unfamiliar.
  • Dimensions can confuse the user with the planes not being controllable until you’ve learned the design layout.
  • Lighting and glow effects are excessively complicated.
  • Most tools limited to what the user knows.
