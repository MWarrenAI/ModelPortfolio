STL Viewer

A STL Viewer allows us to view 3D Models used for CAD Software. Stl files can store attributes such as measurements regarding the 3D Model. This only represents the surface of the object rather than the properties of the object itself.

Mocap

Records the capture of objects and actors in scene and transforms the captured positions of the 3D space into that of controlled 3D Objects replacing.

3D Scanning

The SQLLitePyProject is a short simple Python Program which allows you to view, edit and delete rows from a basic database using sqllite.
