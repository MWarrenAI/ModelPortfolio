3D Systems Touch X Haptic Modelling Arm



Figure 32 Using the Haptic Modelling Arm (Photograph taken by Mark Warren)


The Touch X Modelling Arm applies haptic feedback to simulate a 3d environment of an object – this allows the device to offer movement in 6 dimension of positional sensing and 3 dimensions of freedom force feedback. You can work in a space of around 355mm(width) x 228mm(height) x 180mm(diameter) and a force of around 1.75 Newtons can be applied to the pen in order to achieve results. The pen uses force feedback in order to establish to the user that there is a physical object being drawn upon.



Figure 33 Smoothing Poro's tongue (photo taken by Mark Warren)


Various tools can be used to carve, create mesh objects and modify shapes created using the pen and Geomagic Freeform software.

We found that the pen had limited use for our model – we chose not to make use of the pen since this would have costed us time and money to adapt to the size we wanted to print out: similar issue with the 3D printed head. If given time and money we may have made use of the pen as this would have offered us a means to create our own tone for skin but because of personal preference and tool available in Blender software we found we couldn’t implement changes we would have liked such as easily manipulating the shapes produced as the pen would only be available at Staffordshire University itself thereby meaning the pen is not as portable as we would have liked for the project.


An industry example of the pen would be use for surgical interns to perform accurate surgery using the pen to simulate a medical procedure (Wade, A., 2018).
