Project Previews
Introduction - General brief of project goals and what the group hoped to accomplish.
Plan - What MOSCOW requirements the group agreed upon as well as a general discussion to decide on what tools we should use to accomplish the group's goals.
STL Viewer - A online tool used to display 3 Dimensional models.
Blender - Software used to create Mesh Models and Animations allowing positioning in 3 Dimensional Space.
Fusion 360 - CAD CAM software used to create Models and generate animation previews.
3D Scanning - Scanners used to generate realistic models from physical real life objects.
Laser Cutting - Laser Cutters cut shapes from flatbed material to be used in projects.
Touch X - A pen used to simulate 3 Dimensional space via haptic feedback, this can be used to develop 3 Dimensional models.
Latex - Substance used to create Physical Artifacts and Products.
Mocap - 3 Dimensional Object Manipulation via use of Cameras and Tracking. This Renders Computer Generated Objects in place of trackers found by cameras.
Conclusion - Summary of methods the group used to accomplish task and thoughts on what we would improve/change if we continued the project.
